# ملخص المشروع

## جمعية - منصة الإدخار الجماعي

### نظرة سريعة

**جمعية** هو تطبيق ويب احترافي لنظام الإدخار الجماعي (ROSCA) موجه للسوق الجزائري.

### المعلومات الأساسية

| البند | التفاصيل |
|-------|----------|
| **الاسم** | جمعية |
| **الإصدار** | 1.0.0 |
| **التاريخ** | 2025-02-28 |
| **التقنية** | React + TypeScript + Vite |
| **التصميم** | Tailwind CSS + shadcn/ui |
| **الحالة** | Prototype كامل |

### الملفات الرئيسية

```
├── src/
│   ├── components/
│   │   ├── screens/        # 15 شاشة
│   │   ├── common/         # مكونات مشتركة
│   │   └── ui/            # 40+ مكون shadcn/ui
│   ├── store/             # Zustand store
│   ├── data/              # بيانات وهمية
│   ├── types/             # أنواع TypeScript
│   ├── hooks/             # Hooks مخصصة
│   └── lib/               # دوال مساعدة
├── public/                # ملفات ثابتة
├── docs/                  # توثيق
└── tests/                 # اختبارات
```
