# البيانات الوهمية

هذا المجلد يحتوي على البيانات الوهمية للتطبيق.

## الملفات

- `mockData.ts` - جميع البيانات الوهمية

## الكيانات

### المستخدمون (Users)

8 مستخدمين وهميين بمختلف الحالات:
- موثقون
- قيد المراجعة
- غير موثقين
- مستويات دخل مختلفة

### الجمعيات (Jamiyas)

5 جمعيات وهمية:
- مدخرون السريع (10,000 دج)
- دائرة النمو (25,000 دج)
- النخبة (50,000 دج)
- الادخار المصغر (5,000 دج)
- الصندوق الكبير (100,000 دج)

### العمليات (Transactions)

7 عمليات وهمية:
- مساهمات
- استلام
- رسوم

### الإشعارات (Notifications)

7 إشعارات وهمية:
- تأكيد الحجز
- تذكير بالدفع
- تحقق من الهوية
- اكتمال الجمعية

## الثوابت

```typescript
ACTIVATION_FEE = 2000; // دج
BOOKING_EXPIRY = 48 * 60 * 60 * 1000; // 48 ساعة
MOCK_OTP_CODE = '123456';
```

## الدوال المساعدة

- `formatCurrency()` - تنسيق المبالغ
- `formatDate()` - تنسيق التواريخ
- `formatTime()` - تنسيق الوقت
- `getJamiyaStatusText()` - نص حالة الجمعية
- `getJamiyaStatusColor()` - لون حالة الجمعية
- `getIncomeLevelText()` - نص مستوى الدخل
- `getKYCStatusText()` - نص حالة التوثيق
